# SR INTERNATIONAL LTD. Website

Premium single-page React website for SR INTERNATIONAL LTD., a Tech & Smart Home Installation Service headquartered in Uttara, Dhaka.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React icons

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:5173/`.

## Production Build

```bash
npm run build
```

The build output is generated in `dist/`.

## Repository Upload Notes

Commit the source files, `package.json`, `package-lock.json`, and image assets in `src/assets`.
Do not upload `node_modules/` or `dist/`; they are ignored by `.gitignore`.
